## Læringsmål

Etter denne labøvelsen skal du kunne:

- Beskrive funksjonen til inspirasjonsblokken og dens nøkkelkomponenter, inkludert SV diaphragm-enheten
- Inspisere og bytte SV diaphragm-enheten i henhold til Dräger Evita XL servicemanualen
- Montere respiratorkretser med testlunge og verifisere korrekt drift
- Skille mellom volumkontrollerte og trykkontrollerte ventilasjonsmodi
- Utføre gassleverings-verifisering og alarmtesting på en respirator

---

## Sikkerhetsmerknader

> **ESD-FARE.** Denne labøvelsen innebærer direkte kontakt med elektronikk. Elektrostatisk utladning kan skade sensitive komponenter. Sett opp passende ESD-forholdsregler før arbeidet begynner.

- Bruk ESD-håndleddsstropp koblet til benkens jordingspunkt når du håndterer interne komponenter.
- Ikke slå på strømmen til respiratoren mens interne enheter er eksponert.
- Når du tester med testlunge, kontroller at alle kretstilkoblinger er sikre før du starter ventilasjon.
- Rapporter eventuelle utstyrsskader eller uventet oppførsel til veiledende tekniker.

---

## Oppsett av utstyr

1. Plasser Dräger Evita XL på benken. Inspiser kabinettet for fysisk skade og kontroller at strømkabel og gasstilkoblinger er til stede og uskadet.
2. Kontroller at den fysiske kopien av Dräger Evita XL servicemanualen er tilgjengelig på arbeidsstasjonen din.
3. Finn reservedelen for SV diaphragm-enheten og nødvendig verktøy.
4. Ikke koble til gassforsyning eller testlunge før det angis i prosedyren.

## Prosedyre

### Del 1 — Inspeksjon av inspirasjonsblokk og bytte av SV diaphragm (60 min)

**1.1** Les gjennom kapittelet om *inspirasjonsblokken* i servicemanualen (fysisk kopi i laben) under Kapittel 3 *Reparasjonsinstruksjoner*. Gjør deg kjent med oppbyggingen og funksjonen til inspirasjonsblokken før du begynner praktisk arbeid.

**1.2** Inspiser O2-membranen montert i inspirasjonsblokken på respiratoren i henhold til servicemanualen. Kontroller at membranen ikke er sprukket eller på annen måte skadet. Registrer funnene dine i labnotatboken.

**1.3** Bytt *SV diaphragm-enheten* i henhold til trinn-for-trinn-instruksjonene i servicemanualen. Dokumenter hvert trinn etter hvert som du utfører det.

**1.4** Svar på følgende i labnotatboken: Hvorfor er SV diaphragm-enheten viktig med hensyn til forsyningstrykk-feil eller strømbrudd på respiratoren? (Hint: se avsnitt 3.3 i servicemanualen om utstyrsregisteret.)

---

### Del 2 — Gassleverings-verifisering (45 min)

**2.1** Monter en respiratorkrets med testlunge. Ikke bruk fukter for denne testen.

**2.2** Slå på Evita XL og konfigurer følgende basisinnstillinger:

| Parameter | Innstilling |
|---|---|
| Modus | VC-CMV |
| Tidalvolum (Vt) | 500 mL |
| Respirasjonsfrekvens (RR) | 12 pust/min |
| PEEP | 5 cmH₂O |
| FiO₂ | 0,21 |

**2.3** La respiratoren sykle i minst 2 minutter for å stabilisere seg. Observer bølgeformene på respiratorens display og bekreft normal drift.

**2.4** Test gasslevering ved tre tidalvolum-innstillinger. For hver innstilling, la det gå minst 1 minutt med stabil sykling, og registrer deretter respiratorens display-avlesninger og testlungens respons:

| Tidalvolum-innstilling | Respirator display Vt (mL) | Testlunge-observasjon |
|---|---|---|
| 300 mL | | |
| 500 mL | | |
| 800 mL | | |

**2.5** Kommenter eventuelle avvik mellom innstilte verdier og leverte volumer.

---

### Del 3 — Ventilasjonsmodi og alarmtesting (45 min)

#### 3.1 Sammenligning av ventilasjonsmodi

Gå gjennom betjening og kontroller på Evita XL i henhold til brukermanualen. Test de ulike ventilasjonsmodi tilgjengelig på enheten. For hver testede modus, registrer:

- Modusnavn og forkortelse
- Om modusen er volumkontrollert eller trykkontrollert
- Nøkkelparametere som kreves for den modusen

Fyll ut følgende tabell i labnotatboken:

| Modus | Kontrolltype (Volum/Trykk) | Nøkkelparametere |
|---|---|---|
| VC-CMV | | |
| PC-CMV | | |
| SIMV | | |
| CPAP/ASB | | |
| Andre tilgjengelige modi | | |

#### 3.2 Alarmtesting

Test følgende alarmer og registrer observasjonene dine:

**Frakoblingsalarm:**
1. Med respiratoren i drift i VC-CMV-modus, koble fra pasientkretsen fra testlungen.
2. Mål tiden fra frakobling til alarmaktivering.
3. **Akseptkriterium:** Frakoblingsalarmen skal utløses innen 15 sekunder (IEC 60601-2-12).

**Høytrykksalarm:**
1. Sett høytrykksalarm-grensen til 30 cmH₂O.
2. Delvis okkluder testlungekretsen for å generere forhøyet luftveistrykk.
3. Registrer trykket der alarmen utløses.

**Apnéalarm:**
1. Bytt til en spontanpustemodus (f.eks. CPAP).
2. Ikke simuler spontane pust på testlungen.
3. Mål tiden til apnéalarmen aktiveres.

Registrer alle alarmtestresultater i en oppsummeringstabell.

---

### Del 4 — Diskusjonsspørsmål (30 min)

Svar på følgende spørsmål i labnotatboken. Du vil diskutere svarene dine med gruppen på slutten av øvelsen.

1. Forklar formålet med SV diaphragm og hva som skjer hvis den svikter. I svaret ditt, ta for deg konsekvensene for både forsyningstrykk-feil og strømbrudd-scenarier.

2. Sammenlign volumkontrollerte og trykkontrollerte ventilasjonsmodi — hva er de viktigste forskjellene i hvordan respiratoren leverer pust? Diskuter hvilken parameter som er garantert (volum eller trykk) i hver modus og hva som varierer.

3. Hvorfor har Evita XL både turbin/blåser og tilkoblinger for komprimert gass? Når vil hver av dem brukes? Vurder både normal sykehusdrift og nød-/transportscenarier.

4. IEC 60601-2-12 krever at frakoblingsalarmen utløses innen 15 sekunder. Hvorfor er rask frakoblingsdeteksjon klinisk kritisk? Hva kan skje med en pasient hvis alarmen var forsinket utover denne grensen?

## Krav til labrapporten

Send inn en maskinskrevet labrapport innen datoen angitt i timeplanen. Rapporten skal inneholde:

- En tittelside med navn, studentnummer, emnenummer og dato
- Fullførte inspeksjonsfunn fra Del 1, inkludert tilstanden til O2-membranen og prosedyren for bytte av SV diaphragm
- En tabell for sammenligning av ventilasjonsmodi (Del 3.1) som identifiserer hvilke modi som er volumkontrollerte og hvilke som er trykkontrollerte
- Resultattabell for gassleverings-verifisering (Del 2.4)
- Resultattabell for alarmtesting (Del 3.2), inkludert målte tider og sammenligning mot akseptkriterier
- Skriftlige svar på de fire diskusjonsspørsmålene (Del 4)
- En kort konklusjon (200–300 ord) der du diskuterer de samlede funnene dine og den klinisk ingeniørfaglige betydningen av rutinemessig respiratorvedlikehold
- En referanseliste med henvisninger til IEC 60601-2-12, Dräger Evita XL service- og brukermanualer og alle andre kilder som er brukt
